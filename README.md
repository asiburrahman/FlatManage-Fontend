# 🏢 ManageFlat

**ManageFlat** is a premium, full-featured apartment management platform built with **React**, **Firebase**, **MongoDB**, and **Express.js**. It features a custom-built, high-end design system with seamless **Dark Mode** support, designed to streamline apartment bookings, rent payments, and community management.

🌍 **Live Demo:** [https://manageflat-8f1f4.web.app](https://manageflat-8f1f4.web.app)

---

## ✨ Premium UI & Design System

The latest version of **ManageFlat** introduces a professional-grade UI library built with **CSS Modules** and **Vanilla CSS** for maximum flexibility and performance.

*   💎 **Premium Component Library**: Custom-built `Button`, `Input`, `Card`, `Badge`, `Modal`, `Table`, and `Dropdown` components.
*   🌓 **Native Dark Mode**: Fully integrated theme support with sophisticated color palettes for both light and dark modes.
*   🎨 **Modern Aesthetics**: Glassmorphism effects, smooth gradients, and micro-animations for a premium user experience.
*   ⚡ **TanStack Query**: High-performance server-state management for smooth data transitions.

---

## 🎯 Core Features

✅ **Firebase Authentication** — Secure Login/Signup with Email or Google  
👥 **Role-Based Access Control** — Dedicated dashboards for Admins, Members, and Users  
🏠 **Smart Apartment Listings** — Advanced filtering, responsive 4-card grid, and real-time availability tracking  
🎫 **Coupon Management** — Dynamic coupon system with theme-aware management tools  
💳 **Stripe Integration** — Secure, integrated payment processing with transaction tracking  
📜 **Community Announcements** — Centralized hub for building news and updates  
📊 **Financial Tracking** — Comprehensive payment history and monthly rent summaries  
📲 **Ultra-Responsive** — Optimized for flawless performance across mobile, tablet, and desktop  
📢 **Interactive Feedback** — Integrated SweetAlert2 and Toastify with full theme support

## 🔧 Installation & Setup

### 🔹 Backend Setup
```bash
cd backend
npm install
node index.js
```

### 🔹 Frontend Setup
```bash
cd frontend
npm install
npm run dev
```

## 📦 Tech Stack

| Frontend | Backend |
| --- | --- |
| React 19 + Vite | Node.js + Express.js |
| CSS Modules | MongoDB |
| TanStack Query (React Query) | Firebase Admin SDK |
| SweetAlert2 + Toastify | Stripe API |
| Framer Motion (Animations) | dotenv |
